# 🚀 K-Quest 추가 기능 가이드 (Super App Features)

K-Quest를 단순한 웹사이트가 아닌, **강력한 슈퍼 앱**으로 만들어주는 3가지 핵심 기능을 소개합니다.

---

## 1️⃣ PWA (앱 설치 & 오프라인 지원) 📱

별도의 앱 스토어 다운로드 없이, 웹사이트에서 바로 앱을 설치할 수 있습니다.

### 적용 방법
1. 모바일 브라우저(Chrome, Safari)로 K-Quest 접속
2. **"홈 화면에 추가"** 또는 **"앱 설치"** 버튼 클릭
3. 바탕화면에 K-Quest 아이콘 생성됨
4. 이제부터 **네이티브 앱처럼** 전체 화면으로 실행됩니다!

### 기술적 특징
- **Manifest.json**: 앱 아이콘, 테마 색상, 스플래시 화면 설정 완료
- **오프라인 지원**: 인터넷이 끊겨도 기본 화면이 표시됨 (캐싱)

---

## 2️⃣ 퀵 퀘스트 템플릿 (Quick Templates) ⚡

의뢰자가 무엇을 적어야 할지 고민할 필요가 없습니다. 클릭 한 번으로 전문가 수준의 의뢰서가 완성됩니다.

### 지원 템플릿
- **🍽️ 식당 예약**: 인원, 알러지, 날짜 등 필수 정보 자동 입력
- **🎫 티켓팅/굿즈**: 공연명, 좌석 등급, 예매처 등
- **🚶 로컬 가이드**: 여행 스타일, 이동 수단, 예산 등
- **📦 배달/심부름**: 픽업/배달 장소, 물품 비용 등

### 사용법
`QuestTemplateSelector` 컴포넌트를 퀘스트 작성 페이지 상단에 배치하면 됩니다.

---

## 3️⃣ SOS 안심 버튼 (Safety Beacon) 🚨

오프라인 만남이 불안한 사용자를 위한 최후의 안전장치입니다.

### 기능
- 화면 우측 하단에 항상 떠있는 **붉은색 비상 버튼**
- 클릭 시 **"도와주세요! 현재 위치: [구글맵 링크]"** 문자가 자동으로 작성됨
- 문자 메시지 앱이 즉시 실행되어 지인/경찰에게 전송 가능

### 특징
- **위치 기반**: GPS를 사용하여 현재 내 위치를 정확히 찍어줌
- **즉시성**: 복잡한 절차 없이 클릭 두 번으로 구조 요청
