## 📍 Step 1: DB 설정 (5분)

### 🖱️ 클릭 순서

#### 1번 클릭
- 크롬 켜기
- 주소창에 `supabase.com` 입력
- 로그인하기

#### 2번 클릭
- 화면 왼쪽에 **"SQL Editor"** 라고 적힌 글자 찾기
- 그거 클릭

#### 3번 클릭 (이걸 3번 반복하세요)
1. 폴더에서 `supabase_admin_chat.sql` 파일 열기
2. 전체 선택 (Ctrl+A)
3. 복사 (Ctrl+C)
4. 웹사이트로 돌아가기
5. 큰 빈칸에 붙여넣기 (Ctrl+V)
6. 오른쪽 위에 **"Run"** 버튼 클릭
7. "Success" 라는 초록색 글자 뜨면 성공!

🔁 이제 똑같은 방법으로:
- `supabase_income_tracking.sql` 파일로 반복
- `supabase_error_logs.sql` 파일로 반복

### ✅ 확인
- 왼쪽에 "Table Editor" 클릭
- 표 목록에서 다음 이름들이 보이면 **성공**:
  - admin_support_chats
  - performer_annual_income
  - error_logs
