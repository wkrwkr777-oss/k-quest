# 📘 K-Quest 사장님 운영 매뉴얼

**"하루 30분으로 플랫폼 관리하는 법"**

---

## 🌅 매일 해야 할 일 (10분)

### 1. 대시보드 확인 (5분)
- `/admin` 페이지 접속
- **오늘 매출** 확인
- **미처리 분쟁** 확인 (빨간 숫자가 있으면 즉시 처리)
- **신규 가입자** 수 확인

### 2. 관리자 채팅 확인 (5분)
- 사용자가 문의한 내용이 있는지 확인
- 5분 내로 간단히 답변 (복잡한 건 "곧 답변 드리겠습니다" 후 나중에 처리)

---

## 📅 주 1회 해야 할 일 (20분)

### 1. 전문가 인증 승인 (10분)
- 대시보드 > "인증 승인" 클릭
- AI가 분석한 문서 요약 확인
- 승인/거절 결정

### 2. 분쟁 처리 (10분)
- 양쪽 주장 확인
- 증거 자료 검토 (챗 로그, 사진 등)
- 결정: 환불 / 부분 환불 / 거절

---

## 📊 월 1회 해야 할 일 (30분)

### 1. 매출 분석
- 이번 달 vs 지난 달 비교
- 인기 퀘스트 카테고리 파악
- 전략 수정 (인기 있는 카테고리 홍보 강화)

### 2. 수행자 관리
- 상위 10% 수행자에게 감사 메시지 발송
- 하위 10% 수행자에게 피드백 제공

---

## 🗓️ 연 1회 해야 할 일 (1월)

### 1. 지급명세서 생성 및 제출
- `/api/admin/payment-statement?year=2024` 접속
- CSV 파일 다운로드
- 국세청 홈택스에서 "지급명세서 제출" 메뉴로 업로드
- **기한: 매년 1월 말까지**

### 2. 약관 및 정책 검토
- 이용약관이 현행 법률에 맞는지 확인
- 필요 시 변경 후 사용자에게 공지

---

## 🚨 긴급 상황 대처법

### 1. 서버가 다운됐을 때
- Vercel 대시보드 접속 (vercel.com)
- Deployment 상태 확인
- 문제 있으면 "Redeploy" 클릭

### 2. 결제 오류 발생 시
- Stripe/PayPal 대시보드 접속
- 해당 거래 ID 검색
- "Refund" 처리 후 사용자에게 사과 메시지

### 3. 분쟁이 법적 문제로 비화될 때
- **절대 혼자 해결하려 하지 마세요!**
- 변호사와 상담 (법률 비용은 플랫폼 수익에서 충당)

## 💡 자주 묻는 질문 (FAQ)

### Q. 수행자가 소득세를 안 냈다고 국세청에서 연락 오면?
**A.** "저희는 중개 플랫폼이며, 지급명세서는 제출했습니다. 수행자 개인이 신고할 의무입니다."라고 답변하세요.

### Q. 의뢰자가 환불 요구하는데 수행자는 일 다 했다고 주장하면?
**A.** 증거 자료(챗 로그, 사진)를 보고 판단하세요. 애매하면 50:50 분배가 공정합니다.

### Q. 플랫폼이 너무 커져서 혼자 관리 못하겠어요!
**A.** 축하드립니다! 이제 직원을 고용할 때입니다. "고객지원 매니저"를 먼저 뽑으세요.
