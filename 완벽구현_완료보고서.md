## 🚀 로컬 임시 웹 실행 방법

### 1️⃣ 환경 변수 설정 (필수!)

```bash
# .env.example을 .env.local로 복사
copy .env.example .env.local
```

`.env.local` 파일 수정:
```
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

### 2️⃣ 로컬 서버 실행

```bash
npm run dev
```

### 3️⃣ 브라우저에서 확인

```
✅ http://localhost:3000 - 프리미엄 홈페이지
✅ http://localhost:3000/auth/signin - 로그인
✅ http://localhost:3000/auth/signup - 회원가입
✅ http://localhost:3000/dashboard - 대시보드
✅ http://localhost:3000/quests - 퀘스트 목록
✅ http://localhost:3000/quests/1 - 퀘스트 상세
✅ http://localhost:3000/quests/create - 퀘스트 생성
```
