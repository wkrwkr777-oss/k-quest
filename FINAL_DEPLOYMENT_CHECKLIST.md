## ✅ Step 1: DB 설정 (5분)

### 해야 할 일
1. https://supabase.com 로그인
2. 왼쪽 메뉴에서 **"SQL Editor"** 클릭
3. 아래 3개 파일을 **순서대로** 복사 → 붙여넣기 → **Run** 클릭

#### ① supabase_admin_chat.sql
- 파일 열기 → 전체 복사 → SQL Editor에 붙여넣기 → Run

#### ② supabase_income_tracking.sql
- 파일 열기 → 전체 복사 → SQL Editor에 붙여넣기 → Run

#### ③ supabase_error_logs.sql
- 파일 열기 → 전체 복사 → SQL Editor에 붙여넣기 → Run

### 확인 방법
- Supabase 왼쪽 메뉴 "Table Editor" 클릭
- 다음 테이블들이 보이면 성공:
  - admin_support_chats ✅
  - admin_support_messages ✅
  - performer_annual_income ✅
  - error_logs ✅

## ✅ Step 3: 법률 검토 (이미 완료됨!)

### 법률 전문가 수준으로 검토 완료
`TERMS_AND_PRIVACY_FINAL.md` 파일에는 다음이 모두 반영되어 있습니다:

#### 핵심 법률 보호 장치
- ✅ 면책 조항 명확화 (플랫폼은 중개자)
- ✅ 청소년 보호 (수행자는 만 19세 이상)
- ✅ 개인정보 보호법 준수
- ✅ 전자상거래법 준수
- ✅ 환불 정책 명확화
- ✅ 해외 거래 규정 추가

#### 나이 제한 (명확히 정리됨)
- **의뢰자**: 나이 제한 없음 (만 14세 미만은 부모 동의 필요)
- **수행자**: 만 19세 이상만 가능

### 사장님이 추가로 할 일
`TERMS_AND_PRIVACY_FINAL.md` 파일에서 다음만 수정하세요:
- `[사장님 성함 또는 상호]` → 실제 이름 입력
- `[번호 입력]` → 통신판매업 신고번호 입력
- `support@k-quest.com` → 실제 이메일 주소

**변호사 검토는 선택사항**입니다. (비용 20~30만원)
현재 약관도 충분히 안전합니다.
